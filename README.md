# 学生信息管理系统

基于 Java Swing + JDBC + MySQL 开发的桌面端学生信息管理系统。

## 功能
- 查询所有学生信息（支持按姓名搜索）
- 添加学生
- 修改学生信息
- 删除学生

## 技术栈
- Java Swing（GUI界面）
- JDBC（数据库连接）
- MySQL 8.0（数据存储）

## 开发过程

### v1.0 基础版本
完成项目基本框架搭建：
- 设计数据库表结构（id、姓名、年龄、性别、成绩）
- 实现 DBUtil 数据库连接工具类
- 完成增删改查核心功能
- 搭建 Swing 基础界面

### v1.1 功能升级
- 新增"专业"字段，使用 ALTER TABLE 修改数据库结构
- 新增按姓名模糊搜索功能
- 重构 UI 界面：配色优化、隔行变色表格、圆角卡片表单
- 修复异常处理问题（SQLException → Exception）

## 运行环境
- JDK 8+
- MySQL 8.0
- mysql-connector-java-8.0.26
